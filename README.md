# TaskManagement
A task management app in .NET Core 8.

This solution consists of the following projects
- /Backend - Backend API
- /Backend.Tests - Backend Unit Tests
- /Frontend - Blazor WebAssembly Frontend
- /Frontend.Tests - Frontend Unit Tests
